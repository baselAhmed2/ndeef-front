
  # Laundry Service App UI

  This is a code bundle for Laundry Service App UI. The original project is available at https://www.figma.com/design/vE59i07WsfLJ99Cg5BQhwX/Laundry-Service-App-UI.

  ## Running the code

  Run `npm i` to install the dependencies.

  Create `.env.local` and set:

  `NDEEF_BACKEND_URL=http://ndeefapp.runasp.net`

  `NEXT_PUBLIC_GOOGLE_CLIENT_ID=your-google-client-id.apps.googleusercontent.com`

  In Google Cloud Console, add these Authorized JavaScript origins to the OAuth client:

  `http://localhost:3000`

  `http://127.0.0.1:3000`

  Run `npm run dev` to start the development server.
